üì∑ Barcode Scanning

When you scan a product barcode, we look up nutrition information from the Open Food Facts database (openfoodfacts.org). This is an open, collaborative database.

What happens:

  • The barcode number is sent to Open Food Facts API
  • Nutrition data is returned and displayed
  • No personal information is included in this request
  • Open Food Facts cannot identify you from a barcode lookup

üîê Data Security

Local Storage Security

Your data is stored in your device's secure local storage. This data is:

  • Encrypted by your device's operating system
  • Protected by your device's passcode/Face ID
  • Inaccessible to other apps
  • Deleted when you uninstall the app

Data Transmission Security

When data is transmitted (barcode lookups):

  • All connections use HTTPS encryption
  • Data is encrypted in transit
  • We use industry-standard security protocols
